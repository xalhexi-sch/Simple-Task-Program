# Note Management System

A simple Python script to manage text-based notes, with functionality to add, list, read, and delete notes. Notes are saved as `.txt` files in a designated directory (`data/notes`).

## Features

- **Add Note**: Create a new note with a title and content. The note is saved as a `.txt` file with a timestamp of when it was created.
- **List Notes**: View a list of all saved notes by their title.
- **Read Note**: Read the content of a specific note by its title.
- **Delete Note**: Delete a note by its title.

## Requirements

- Python 3.6 or later
